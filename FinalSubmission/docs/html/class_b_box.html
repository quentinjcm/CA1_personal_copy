Detailed Description

The BBox class holds the data for bounding boxes that contain triangle meshes. They also implement the ray/bounding box intersection algorithm to speed up intersection checks for rays. If a ray doesnt intersect the bbox, then it is not checked for intersections with the contained triangles.
