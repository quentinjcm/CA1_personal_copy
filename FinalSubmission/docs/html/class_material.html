Detailed Description

The Material class defines a material that is appled to objects in the scene. It contains the texture file and atributes that define how the geometry is shaded.

Parameters
[in]_colouris the default diffuse colour of the material, to be used if no texture file is found
[in]_smoothnessis the smootheness of the material used for phong shading
[in]_IORis the index of refraction, which defines how much light is reflected and refracted at different angles
[in]_isTransparentis a bool that tells the renderer to send difracted rays through the object
[in]_isReflectiveis a bool that tells the renderer to send reflected rays into the scene to get reflected colour
[in]_texFileis a string that is the relative path to the image file that is the diffues texture for the material
