Detailed Description

the Scene parser reads in scene files as .txts and translates them into an objects to be placed in the scene. The class takes a pointer to its scene at constructio instead of creating its own so that there is a potenial to have several files parsed to the same scene object. For a guide on formatting scene files please see the readme. The parser uses the boost spirit library to define the scene file syntax in EBFN. The parser holds a pointer toa current material, which is passed to each mesh as it is constructed. see: https://en.wikipedia.org/wiki/Extended_Backus%E2%80%93Naur_Form
