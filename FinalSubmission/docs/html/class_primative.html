Detailed Description

The primatve class in an abstract base class that all primatives in the scene should inherit from. This gives them their basic interface of intersection methods and a function to extract their opengl data. The idea was to ipmlement two primatives which inherit from this, the aggregate primative and the geometric primative. The aggragate primative was to hold other primatives in acceleration structures and the geometric primative holds triangle meshes.While i didnt have time to implement an aggregate primative the setup allows that to happen without too much dificulty and andl enables the easy expansion to othert types of primatives such as nurbs curves and volumes.
